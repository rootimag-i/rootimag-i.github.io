Linux 快捷键大全

常用的命令

  • ls：列出目录内容，详见Linux ls 命令
  • cd：切换目录，如 cd..（返回上级目录）、cd ~（返回用户主目录）。
  • pwd：显示当前目录。
  • mkdir：创建目录。
  • rmdir：删除空目录。
  • rm：删除文件，rm -r 可递归删除目录及其内容。
  • cp：复制文件，cp -r 可递归复制目录。
  • mv：移动文件，也可用于重命名文件，如 mv old_name new_name。
  • cat：显示文件内容。
  • grep：搜索文件内容。
  • find：搜索文件。
  • chmod：修改文件权限，数字表示法如 chmod 755 file，符号表示法如 chmod u+x file 给文件所有者添加可执行权限。
  • chown：修改文件所有者。
  • chgrp：修改文件所属组。
  • 当前用户、当前终端运行进程:ps
    ps-a系统所有用户和终端运行进程,不包括shell
    ps-fa格式化输出所有用户和终端运行进程,用户名用UID替代CMD列显示完整的命令行
    ps-au显示所有用户和进程,包括系统任务(多数不与终端关联
  • kill：终止进程，kill -9 可强制终止进程。
  • top：显示系统进程，按 q 可退出 top 界面。
  • df：显示磁盘空间。
  • du：显示目录或文件所占用的磁盘空间。
  • free：显示内存使用情况。
  • ifconfig：显示网络接口信息（在一些新系统中被 ip 命令替代）。
  • ping：测试网络连接。
  • wget：下载文件。
  • curl：下载文件。
  • tar：压缩文件，常用选项如 tar -czvf（创建并压缩为 gzip 格式的归档文件）、tar -xzvf（解压 gzip 格式的归档文件）。

常用的文件操作

  • touch：创建文件。
  • cat：显示文件内容。
  • grep：搜索文件内容。
  • find：搜索文件。
  • rm：删除文件。
  • cp：复制文件，cp -r 可递归复制目录。
  • mv：移动文件，也可用于重命名文件。

常用的系统管理命令

  • ps：显示进程状态，ps -aux 显示所有用户的所有进程。
  • kill：终止进程，kill -9 可强制终止进程。
  • top：显示系统进程，按 q 退出 top 界面。
  • df：显示磁盘空间。
  • du：显示目录或文件所占用的磁盘空间。
  • free：显示内存使用情况。
  • ifconfig：显示网络接口信息（在一些新系统中被 ip 命令替代）。
  • ping：测试网络连接。
  • wget：下载文件。
  • curl：下载文件。
  • tar：压缩文件，tar -czvf 创建并压缩为 gzip 格式的归档文件，tar -xzvf 解压 gzip 格式的归档文件。
  • unzip：解压文件。
  • useradd：添加用户。
  • userdel：删除用户。
  • groupadd：添加用户组。
  • groupdel：删除用户组。
  • service：用于管理系统服务（不同系统略有差异，如 service httpd start 启动 Apache 服务）。
  • systemctl：在较新的 Linux 系统中用于管理服务，如 systemctl start nginx 启动 Nginx 服务。

常用的网络命令

  • ifconfig：显示网络接口信息（在一些新系统中被 ip 命令替代）。
  • ping：测试网络连接。
  • wget：下载文件。
  • curl：下载文件。
  • netstat：显示网络连接、路由表等信息（在一些系统中已被 ss 命令替代）。
  • ss：更高效地显示网络连接等信息。
  • traceroute：追踪数据包的路由路径。
  • ssh：远程登录到其他主机，如 ssh user@host。

vim 编辑器

插入模式

  • 按 i 进入插入模式，按 Esc 或者 Ctrl + [ 可退出插入模式。

命令模式

  • 按 esc 进入命令模式。
  • 常用操作：
    • 复制：yy 复制当前行，nyy 复制 n 行。
    • 粘贴：p 粘贴到光标后，P 粘贴到光标前。
    • 删除：dd 删除当前行，ndd 删除 n 行。

跳转模式

  • 按 g 跳转到文件开头。
  • 按 G 跳转到文件结尾。
  • 按 0 跳转到行首。
  • 按 $ 跳转到行尾。
  • 按 ^ 跳转到行首的第一个非空白字符。

末行模式

  • 按 : 进入末行模式。
  • 常用操作：
    • :w：保存文件。
    • :q：退出编辑器。
    • :wq：保存并退出。
    • :set nu：显示行号。
