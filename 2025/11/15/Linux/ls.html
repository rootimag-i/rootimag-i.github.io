Linux ls 命令

Linux ls（英文全拼： list directory contents）命令用于显示指定工作目录下之内容（列出目前工作目录所含的文件及子目录)。

语法

ls [-alrtAFR] [name...]

参数

参数 说明
-a -all 显示所有文件，包括隐藏文件。
-l --long 以详细格式显示文件和目录的详细信息。
-A --almost-all 显示除 . 和 .. 之外的所有文件。
-h --human-readable 以人类可读方式显示文件大小。
-t --time 按修改时间排序。
-r --reverse 反向排序。
-S --size 以文件大小排序。
-F --classify 在每个文件名后显示类型指示符。
-R --recursive 递归列出所有目录下的内容。
--color 控制是否使用颜色显示输出。
--group-directories-first 按字母顺序排列目录，然后按文件名排序。
-d --directory 仅显示目录，不包括其他文件。
-i --inode 显示inode信息。
-n --numeric-uid-gid 以数字形式显示 UID 和 GID（替代用户名和组名）。
-o --octal 以八进制显示文件权限。
-m 以逗号分隔的列表形式显示文件。
-1 每行只显示一个文件。
-Q --quote-name 在文件名中使用单引号。
-Z --context 显示文件安全上下文。
-v --version 显示版本信息。
--help 显示帮助信息。

文件类型

符号 说明
- 普通文件
d 目录
l 符号链接
b 块设备文件
c 字符设备文件
p 管道文件
s 套接字文件

访问权限

符号 说明
r 读权限
w 写权限
x 执行权限
- 无权限
